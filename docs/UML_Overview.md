```plantuml
Object -> AnotherObject : call
AnotherObject --> Object : return
deactivate Object
```

### Activity Diagram
```plantuml
start
:Activity;
if (condition?) then (yes)
    :Action A;
else (no)
    :Action B;
endif
stop
```

### Use Case Diagram
```plantuml
actor Actor
usecase "Use Case" as UC
Actor --> UC
UC ..> UC2 : <<include>>
UC <.. UC3 : <<extend>>
```
